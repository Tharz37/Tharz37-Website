Gesture Controlled Home Automation

Controlling Hardware with a Wave

This was another project from my first year, built alongside the light follower. This time, it was a full team effort. We wanted to move beyond simple sensors and try something that felt like magic: computer vision.

The "Mini AI Home" concept was to control a model house just by showing gestures to a camera. Using Python with OpenCV and MediaPipe, the system identifies different hand poses in real-time and sends commands over a serial port to an Arduino Uno, which then controls the lights, a fan, and a buzzer.

Features and Technical Setup

The logic was split: Python handled the "seeing" and gesture logic, while the Arduino handled the "doing" (hardware control). Here is the complete breakdown of the features, hardware, and gesture list we implemented.

Features
--------
- Open/Closed Palm: Controls the LED (light)
- Thumbs Up/Down: Controls the buzzer (sound)
- Victory/One Finger: Controls the fan (motor)
- Intensity Control: Adjusts brightness/speed via thumb/index distance.

Hardware & Pinout
-----------------
- Arduino Uno
- LED:           Pin 3
- Fan (Motor):   Pin 5
- Buzzer:        Pin 6

Software & Setup
----------------
1. Install Python libraries:
   pip install opencv-python mediapipe
2. Upload the .ino sketch to the Arduino Uno.
3. Modify the serial port (e.g., COM3) in the Python script to match your Arduino.
4. Run the Python script to start gesture recognition:
   python gesture_control.py

A Foundation in CV

This project was a fantastic, hands-on introduction to computer vision. It was a great team effort and taught us how to build a full-stack system, from the camera input (Python/OpenCV) right down to the hardware output (Arduino/Serial).

It was amazing to see the code instantly recognize a thumbs-up and trigger a physical buzzer. This project gave us a solid foundation for more complex AI and robotics work later on.
