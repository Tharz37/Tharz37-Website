NeuraHome: Gesture and Natural Voice Control

No Wake Word, Just Real Conversation

Most "smart" homes aren't very smart. You have to use a specific app or shout a specific "wake word" like "Hey..." to get anything done. We wanted to build something better.

Our idea for NeuraHome was simple: create an automation system that understands you naturally. You can control it with a wave of your hand, or just by talking. The best part? No special keywords. Just talk, and it understands.

The Tech: Gemini, OpenCV, and ESP32

The magic behind the "no wake word" feature is the Gemini API. The system actively listens, and instead of waiting for a keyword, it processes the natural language to understand the *intent*. This means "illuminate the room," "lights on," and "bright the room" all correctly trigger the same "LED ON" command.

Simultaneously, a Python script using OpenCV and MediaPipe watches a camera feed. It recognizes specific hand gestures to control the fan, LED lights, and even video playback. All these commands are sent via WebSockets to an ESP32, which acts as the central hub, controlling the hardware and updating a live web interface with the device status.

Technical Details & Setup

For those interested in the architecture, here is the complete project structure and setup guide.

Project Structure
-----------------
Home Automation/
├── Main_code/      # Main Arduino code for ESP32
│   └── website/    # Web interface for control and status
├── CVandVoice/     # Code for gesture and voice control
└── WS_test/
    ├── esp32/      # WebSocket test code for ESP32
    └── Python/     # WebSocket test code for Python

Requirements
------------
Hardware:
- ESP32
- Microphone
- Camera

Software:
- Arduino IDE
- Python 3.x
- Libraries:
  - pip install opencv-python
  - pip install mediapipe
  - pip install SpeechRecognition
  - pip install google-generativeai
  - pip install websocket-client
  - pip install requests
  - pip install pyaudio

Setup Instructions
------------------
1. ESP32 Setup:
   - Open Main_code/Main_code.ino and upload to your ESP32.
   - Ensure ESP32 is on the same Wi-Fi as your computer.

2. Python Setup:
   - Install all libraries from the list above.
   - Set your Gemini API key in the Speech.py file.
   - Run Speech.py for voice control.
   - Run openCV.py for gesture control.

3. Web Interface:
   - Open website/web.html in a browser to control and view status.
