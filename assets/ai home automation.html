Mission Objective

This mission focused on bridging the gap between Computer Vision (CV) and physical hardware execution. The primary goal was to bypass traditional physical switches by implementing an "air-interface" capable of identifying human hand poses in real-time to control household subsystems.

Logic Architecture

The system utilizes a Python-based processing hub running MediaPipe to calculate hand landmarks. These coordinate landmarks are then filtered through a gesture-recognition algorithm that translates visual data into specific binary or PWM commands sent over serial communication to an Arduino Uno hub.

Functional Protocols

The system was configured with four primary execution protocols:

  • Illumination: Open palm pose triggers Pin_03.
  • Climate: Radial Victory pose activates PWM-controlled fan motor.
  • Security: Thumbs-up pose initiates the high-frequency buzzer.
  • Intensity: Thumb-to-Index distance modulates output brightness/speed.

Final Summary

Successful deployment of the "Mini AI Home" verified the stability of Serial-to-CV interfacing. This foundation led to the development of higher-level projects involving intent detection and voice-CV hybrid systems (NeuraHome).
